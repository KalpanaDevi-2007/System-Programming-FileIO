# System Programming - File I/O

## Objective

Demonstrate file handling using C++.

## Features

- Create a file
- Write data
- Read data
- Handle file errors

## Compilation

```bash
g++ file_io.cpp -o file_io
```

## Run

```bash
.\file_io.exe
```

## Time Complexity

Writing : O(n)

Reading : O(n)
